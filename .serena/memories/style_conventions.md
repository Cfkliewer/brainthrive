# Style and Code Conventions

## Code Style
- TypeScript with strict typing
- React functional components with hooks
- 'use client' directive for client-side components
- Descriptive component and variable names

## CSS/Styling
- Tailwind CSS utility classes
- Custom medical-focused color palette
- Responsive design with mobile-first approach
- Medical-grade accessibility standards

## File Naming
- kebab-case for directories and files
- PascalCase for React components
- camelCase for variables and functions

## Component Structure
- Interface definitions at top
- State and effects after interfaces
- Helper functions before return
- JSX with proper indentation and organization

## Accessibility Standards
- WCAG 2.1 AA compliance required
- High contrast ratios (4.5:1 minimum)
- Proper focus indicators
- Semantic HTML structure
- Alt text for all images
- Skip navigation links
