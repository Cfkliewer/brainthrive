# BrainThrive Project Overview

## Purpose
BrainThrive is a medical website for a photobiomodulation therapy clinic specializing in brain health treatments. The site promotes advanced light therapy treatments for conditions like ADHD, concussion recovery, mental wellness, and other neurological issues.

## Tech Stack
- **Framework**: Next.js 15.5.2 with App Router
- **UI Framework**: React 19.1.0 
- **Styling**: Tailwind CSS v4 with PostCSS
- **Fonts**: Ubuntu and Ubuntu Mono from Google Fonts
- **Language**: TypeScript
- **Build Tool**: Turbopack (via `--turbopack` flag)

## Key Features
- Medical-focused design with high contrast colors
- Responsive design for mobile and desktop
- Service pages with detailed information
- FAQ sections with interactive accordions
- Animated service carousel
- Professional medical branding
